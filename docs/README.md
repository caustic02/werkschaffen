# werkschaffen.com

Personal practice website for TEDE Holdings LLC d/b/a werkschaffen.

## Stack
- Next.js 15 (App Router)
- Tailwind CSS
- Vercel (hosting)
- Cloudinary (images)
- Resend (email)
- Cal.com (booking)

## Repo structure
- src/app/ — pages and routes
- src/components/ — shared components
- src/lib/ — utilities, config
- docs/ — project documentation
- public/ — static assets
